# Claude/Agent Skills for Claude 4.x

These are Claude-compatible skills for [Godot Engine](https://godotengine.org/) 4.x. It's information distilled from now-current [4.6.x docs](https://docs.godotengine.org/en/stable/).

## Implementation

They are split into two distinct skill sets: one for GDScript-specific details, and the other for C#. 

Personally I just install one or the other; I'd be surprised if installing both would be confusing, but that's for you to discover. ;)

These have been tested with Claude and [OpenCode](https://opencode.ai/) (Using Gemma4 locally.) 

Any code agent that understands Claude-compatible skills should, in theory, work fine.

## Scripts

There are distillation scripts created by Claude present in the root of each section; these are included here for future updates.
